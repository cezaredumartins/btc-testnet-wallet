# Bitcoin Testnet Wallet Generator

Uma carteira HD para Bitcoin Testnet baseada em Node.js, com suporte a:

- Geração de frases mnemônicas (BIP39)
- Derivação de endereços de depósito (P2PKH) via BIP44
- Assinatura e criação de transações de envio

## Instalação

```bash
git clone https://github.com/cezaredumartins/Desafio.dio
cd btc-testnet-wallet
npm install
